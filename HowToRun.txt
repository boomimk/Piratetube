# COPY-PASTE GUIDE FOR PIRATETUBE DOCKER

## STEP 1: Put these 3 files in your folder
Save these files to: C:\Users\aviro\Desktop\FU\Python\

1. Piratetube-Docker.py
2. Dockerfile
3. requirements.txt

## STEP 2: Open PowerShell in that folder
Right-click in the folder â†’ "Open in Terminal" or "Open PowerShell here"

## STEP 3: Build the Docker image
Copy and paste this command:

```powershell
docker build -t piratetube .
```

Wait for it to finish (takes 1-2 minutes first time)

## STEP 4: Run the container
Copy and paste this command:

```powershell
docker run -it --rm -v C:\Users\aviro\Downloads:/downloads piratetube
```

## THAT'S IT!
Videos will now save to: C:\Users\aviro\Downloads

---

## To run it again later:
Just use the Step 4 command again:
```powershell
docker run -it --rm -v C:\Users\aviro\Downloads:/downloads piratetube
```

---

## What the command does:
- `-it` = Interactive mode (you can type)
- `--rm` = Auto-delete container when you quit
- `-v C:\Users\aviro\Downloads:/downloads` = Saves videos to YOUR Downloads folder
- `piratetube` = The name of your Docker image
